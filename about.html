About our Project

Purpose

Air quality affects our daily lives, even when we are unaware of it. This project is intended to raise awareness of air quality issues and promote our users to make informed decisions about their outdoor activities, health, and overall lifestyle. Staying informed about air quality is both convenience but also safety. Unsafe conditions can lead to various health conditions, like respiratory issues.

Our application

This web application will allow users to input their geographic location such as their city, state or zip code using data from reliable APIs to deliver accurate and up-to-date air quality information for a particular area. The data will be presented using an interactive map, as well as using charts as a visualization showing pollutant levels. Our system will also help display tips for health safety based on the air quality index.

APIs

This is the list of APIs that our application uses

  • IQ Air
    • Provides access to real time air quality data for different locations around the world. Iq air also gives coordinates to help make a visual map.
  • Leaflet JS
    • Leaflet is the leading open-source JavaScript library for mobile-friendly interactive maps. This will be used to creative an interactive map that updates with the user's zip code.
  • ZipCodeAPI
    • This is the api used to convert zipcode information entered by the user to city and state data.
